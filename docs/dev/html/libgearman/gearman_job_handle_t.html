Job handle (gearman_job_handle_t)¶

SYNOPSIS¶

#include <libgearman/gearman.h>

gearman_job_handle_t¶

DESCRIPTION¶

A gearman_job_handle_t represents a “job handle”. A job handle is a text string that represents the name of a task (in truth gearman_job_handle_t is a typedef’ed char[GEARMAN_JOB_HANDLE_SIZE]).

gearman_client_job_status() use handles to find the status of tasks. When passed to gearman_client_do_background() it will be populated with the job handle.

RETURN VALUE¶

None.

HOME¶

To find out more information please check: http://gearman.info/
